# Velvety Analytics
Velvety Analytics is a production ready subscription architecture built with React, TypeScript, Bun, Express, Stripe, and Firebase.
It demonstrates secure authentication, Stripe subscription billing, webhook driven lifecycle management, tier based access control, protected backend routes, and real time subscription state synchronization.
Frontend deployed on Netlify. Backend deployed on Render.
Designed to reflect stable SaaS billing infrastructure built for real world usage.
